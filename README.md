# 🦎 Gekko Finance Intelligence

> "Greed, for lack of a better word, is good." — Gordon Gekko

A professional-grade financial intelligence platform with real-time portfolio tracking, 
quantitative analysis, investment banking models, and an interactive AI advisor that teaches as it advises.

## Features
- ◈ Live portfolio dashboard with real-time prices
- 📈 Performance tracking & returns analysis  
- 🧮 Quant metrics: Sharpe, Beta, VaR, Correlation Matrix, DuPont, Horizontal Analysis
- 🎲 Monte Carlo stress testing
- 🏦 Full IB suite: DCF, Comps, Precedent Transactions, LBO, Ratio Analysis
- 📁 Financial statement fetcher + CSV/Excel upload
- 🤖 Interactive AI Advisor (powered by Claude) — knows your portfolio, teaches as it advises

## Deploy to Streamlit Cloud (Free)

### Step 1 — GitHub
1. Go to github.com → Click "+" → New Repository
2. Name it: `gekko`
3. Make it Public
4. Click "Create repository"
5. Upload both files: `app.py` and `requirements.txt`
6. Click "Commit changes"

### Step 2 — Streamlit Cloud
1. Go to share.streamlit.io
2. Sign in with GitHub
3. Click "New app"
4. Repository: `your-username/gekko`
5. Branch: `main`
6. Main file path: `app.py`
7. Click "Deploy!"

### Step 3 — Add AI Advisor
1. Get free API key at console.anthropic.com
2. In the app sidebar → "AI Advisor Key" → paste your key
3. Go to the AI Advisor tab and start chatting!
